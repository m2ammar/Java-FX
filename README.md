# JavaFX Projects

This repository contains **JavaFX GUI-based projects**, including a **Calculator** and various practice exercises, built as part of learning Java and JavaFX fundamentals.

## Features

### Calculator
- Fully functional **GUI calculator** built using JavaFX.
- Supports basic arithmetic operations:
  - Addition (+)
  - Subtraction (-)
  - Multiplication (*)
  - Division (/)
- Clear and intuitive user interface.

### Practice Exercises
- Contains multiple **Java practice programs** for learning:
  - Loops, conditions, and basic logic
  - File handling practice
  - Additional mini JavaFX GUI experiments

## Technologies Used
- **Java 17+**
- **JavaFX 20+**
- **IntelliJ IDEA** for development
